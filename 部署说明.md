## GitHub Pages 部署指南

## 部署步骤

### 第一步：在 GitHub 创建仓库

1. 打开 https://github.com/new
2. 仓库名填：`ecommerce-ai-demo`（或任意名称）
3. 勾选 **Public**（GitHub Pages 免费版仅支持公开仓库）
4. 点击 **Create repository**

### 第二步：推送文件

打开终端（Terminal），依次执行以下命令：

```bash
cd /Users/zhangyue565/.qoderwork/workspace/mq2evcb0t4pgkrtd/outputs/github-pages-deploy

git init
git add index.html
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/ecommerce-ai-demo.git
git push -u origin main
```

把 `你的用户名` 替换成你的 GitHub 用户名。

### 第三步：开启 GitHub Pages

1. 打开仓库页面，点击 **Settings**
2. 左侧菜单找到 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. Branch 选 **main**，目录选 **/ (root)**，点击 **Save**

### 第四步：访问

等待 1-2 分钟后，访问：

```
https://你的用户名.github.io/ecommerce-ai-demo/
```

把这个链接分享给别人即可公网访问。
